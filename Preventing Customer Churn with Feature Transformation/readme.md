# Preventing Customer Churn with Feature Transformation 📡

## Overview
This project focuses on predicting customer churn using Machine Learning and improving model performance through **Feature Engineering** and **Feature Selection** techniques.

The notebook demonstrates how transforming raw features into meaningful insights can significantly improve churn prediction accuracy.

---

## Project Objectives
- Understand customer churn prediction
- Build a baseline ML model
- Apply feature engineering techniques
- Perform feature selection
- Compare model performance before and after transformation
- Analyze the impact of engineered features
